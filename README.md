# ГК ХОСТ. Автотесты на Java

<a id="console"></a>
## Запуск тестов из консоли :computer:

```bash
mvn clean test

```

> `clean` - удаление артефактов предыдущей сборки
> 
> `test` - запуск всех тестов из testng.xml
